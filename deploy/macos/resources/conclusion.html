Installation Complete

The Stem has been installed and the background service is running.

Access the Web UI:

https://localhost:8444 (self-signed cert)

To trust the self-signed cert system-wide, run:

sudo /usr/local/stem/stem install-ca

CLI Usage:

/usr/local/stem/stem version
/usr/local/stem/stem --help
/usr/local/stem/stem tui

Service Management:

sudo launchctl unload /Library/LaunchDaemons/com.stem.plist  # Stop
sudo launchctl load /Library/LaunchDaemons/com.stem.plist    # Start

Tip: Add /usr/local/stem to your PATH for convenience.
